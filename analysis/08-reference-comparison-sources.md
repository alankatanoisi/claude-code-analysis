## 1. Chapter Guide

This chapter only records the external public sources used for comparison analysis, to avoid mixing external materials with source code evidence.

## 2. Codex

- OpenAI Help Center: Using Codex with your ChatGPT plan: <https://help.openai.com/en/articles/11096431>
- OpenAI Help Center (French mirror, for cross-referencing paragraph differences): <https://help.openai.com/fr-fr/articles/11369540-utiliser-codex-avec-votre-offre-chatgpt>
- OpenAI Developers Codex documentation entry point: <https://developers.openai.com/>

Judgments supported:

- Codex has covered CLI, IDE extensions, web, app, SDK, Slack, and other entry points
- Codex distinguishes between local and cloud permission/governance boundaries
- Codex is a coding agent product line covering both local and cloud multi-entry points

## 3. Gemini CLI

- Gemini CLI official repository README: <https://github.com/google-gemini/gemini-cli>

Judgments supported:

- Gemini CLI is a high-standard baseline for public open-source CLI agents
- Its public capabilities include built-in tools, MCP, checkpointing, sandboxing, trusted folders, telemetry

## 4. Aider

- Aider homepage: <https://aider.chat/>
- Aider documentation overview: <https://aider.chat/docs/>

Judgments supported:

- Aider emphasizes terminal pair programming
- Aider emphasizes repo map, git, lint/test feedback loop
- Aider is more lightweight, with lower platformization than the current project
